# 🎨 Anime Face Generation with DCGAN

A from-scratch implementation of **Deep Convolutional Generative Adversarial Networks (DCGAN)** trained on 63,565 anime face images to synthesise novel, photorealistic anime characters.

## 🧠 Architecture

### Generator
```
Latent Vector z (300-dim)
      │
      ▼
Dense(8×8×512) → Reshape(8,8,512)
      │
      ▼
ConvTranspose2D(256) → BN → ReLU   # 8  → 16
ConvTranspose2D(128) → BN → ReLU   # 16 → 32
ConvTranspose2D(64)  → BN → ReLU   # 32 → 64
Conv2D(3, sigmoid)                 # Output: 64×64×3
```

### Discriminator
```
Input: 64×64×3 image
      │
Conv2D(64,  stride=2) → LeakyReLU          # 64 → 32
Conv2D(128, stride=2) → BN → LeakyReLU     # 32 → 16
Conv2D(256, stride=2) → BN → LeakyReLU     # 16 → 8
Conv2D(512, stride=2) → BN → LeakyReLU     # 8  → 4
Flatten → Dense(1, sigmoid)
```

## 🔬 Key Design Choices

| Choice | Reason |
|---|---|
| Strided convolutions (no pooling) | Standard DCGAN — preserves spatial information better |
| BatchNormalization in both nets | Critical for training stability in deep GANs |
| LeakyReLU (α=0.2) in discriminator | Prevents dead neurons vs standard ReLU |
| Label smoothing (real → 0.95) | Reduces discriminator overconfidence |
| Adam(β₁=0.5) | Recommended for GANs — faster oscillation dampening |
| Matched lr (5e-5) for G and D | Prevents one network dominating early |

## 📊 Evaluation

GANs cannot be evaluated with traditional metrics. This project uses:

1. **Qualitative:** Visual inspection of generated image grids across epochs
2. **Latent interpolation:** Smooth transitions between two random latent vectors confirm continuous representation learning
3. **Pixel statistics:** Per-channel μ/σ comparison between real and generated images
4. **FID (offline):** Fréchet Inception Distance for rigorous quantitative evaluation

> Typical DCGAN FID on anime datasets: 80–150. StyleGAN2 achieves ~2–5.

## 🚀 Future Work

- **WGAN-GP** — Wasserstein loss + gradient penalty for more stable training
- **Progressive GAN** — Grow resolution from 4×4 → 256×256
- **Conditional GAN** — Control hair colour, eye type via class conditioning
- **Self-Attention (SAGAN)** — Better long-range spatial coherence
- **Automated FID tracking** — `torch_fidelity` pipeline every 10 epochs
